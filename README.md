# Archiv
Archiv is a simple self hosted file storage solution made using Flask.
It allows you to upload, view and manage your files through a web UI.

## Features

- Upload files through web UI.
- Easy to use interface.
- View files directly in the browser.
- Self hosted.

## Installation
1. Clone this repository.
    ```bash
    git clone https://github.com/nishindudu/Archiv.git
    ```

2. Open the project directory.
    ```bash
    cd Archiv
    ```

3. Install the required dependencies.
    ```bash
    pip install -r requirements.txt
    ```

4. Run the Flask application.
    ```bash
    python app.py
    ```
> <b>NOTE</b><br>
> This will start a development server. For production, consider using a production server like Gunicorn.

5. Open your web browser and navigate to `http://localhost:5000` to access the Archiv web interface.
